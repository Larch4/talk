# 视线要求
设计试点确定后, 就要进行地面起坡计算. 首先要确定**每排视线升高值 $C $**. $C$值为后排观众的视线与前排观众眼睛之间的视高差, 一般定为 $120mm$. 当**错位排列**(后排人的视线擦过前面隔一排人的头顶)时, $C$值取 $60mm$ ; 当**对位排列**(即后排人的视野擦过前排人的头顶)时, $C$值取 $120mm$.

# 房间的净高和层高 (P112)
**房间的净高**是指楼地面面层(完成面)到结构层(梁、板)底面或顶棚下表面之间的垂直距离,其数值反映了房间的高度.
**房间的高度**由**房间的使用性质**、**家具设备的使用要求**、**室内的采光和通风**、**技术经济条件**、**室内空间比例**等几个方面决定的.

# 天然地基与人工地基
### 天然地基
- **天然地基**具有足够的承载力, 不需要经过人工加固, 可直接在其上建造房屋的土层
### 人工地基
- **人工地基**指当土层承载力较差或虽然土层较好, 但上部荷载较大时, 为使地基具有足够的承载能力, 应对土体进行人工加固.
- 人工地基的处理方法有**压实法**、**换土法**、**打桩法**三大类. (P138)

# 基础的买只深度及其影响因素 (P140)
### 基础的埋置深度
基础埋深不能过浅, 不应小于 $0.5 m$, 以防建筑物荷载将基础四周土壤挤出, 或地面受到雨水冲刷、机械破坏等而导致基础暴露, 影响建筑物的安全. 
### 影响基础埋深的主要因素
1. 工程地质条件的影响
2. 地下水影响
3. 土的冻结深度的影响
   土的冻结深度即**冰冻线**. 基础原则上应埋在冰冻线以下 $200 mm$处.

# 按所用材料及受力特点分类
### 无筋扩展基础 (刚性基础)
常见的无筋扩展基础有**混凝土基础**、**毛石混凝土基础**、**砖基础**、**毛石基础**、**灰土基础**、**三合土基础**.
#### 混凝土及毛石混凝土基础
混凝土基础坚固、耐久度高、抗水冻, 多用于地下水位较高或有冰冻情况的建筑.
#### 砖基础
砖基础简便、价格低廉, 但因其强度、耐久性、抗冻性和整体性均较差, 一般多用于5层以下的砖混结构房屋.
#### 毛石基础
毛石基础的强度、抗水、抗冻、抗腐蚀性较好, 但体积较大、自重大、操作要求高、劳动强度大, 多用于临近山区石材丰富的一般标准的砖混结构的建筑基础工程中.

# 预制楼板的类型(P195)
- 分为**普通钢筋混凝土楼板**和**预应力钢筋混凝土楼板**两大类.
- 预制钢筋混凝土楼板的截面形状主要有**实行平板**、**空心板**、**槽型板**三种.
### 实心平板
板面平整、隔声差, 常用于过道, 小开间的房间、阳台、雨蓬、管道盖板或搁板等. 
### 槽形板
一种梁板结合预制构件, 通过在板两侧设置边肋, 使作用在板上的荷载传递给边肋, 从而减少板的计算跨度, 减薄板得厚度. 为了提高板的刚度和便于搁置, 常将板的两端以端肋封闭. 
### 空心板
一种梁板结合预制构件， 受力合理. 根据板内抽孔形状不同, 空心板可分为方孔板、椭圆孔板和圆孔板. 

|      | 实心平板 | 槽型板 | 空心板 |
| :----: | :-------: | :------: | :------: |
| 跨度 | $< 2.4 m$ | $3 \sim 7.2 m$ | $7.2 m$ |
| 板厚 | $50 \sim 80 mm$ | $500 \sim 1200 mm$ | 与楼板的长度有关, 一般: $120 \sim 240 mm$ |
| 板宽 | $600 \sim 900 mm$ | $25 \sim 30 mm$ | $500 mm、600 mm、900 mm、1200 mm$ |
| 肋高 |  | $120 \sim 300 mm$ |  |
| 备注 |  | 当跨度达 $6 m$ 时, 应在板中每隔 $500 mm \sim 700 mm$ 增设一道横肋 |  |

# 楼地面防水构造
### 防水构造
对防水要求高的房间, 可增设防水附加层. 为防止四周与墙相交处渗漏, **防水层应向上延伸至少 $150 mm$**. **当遇到开门时, 防水层应向上延伸 $250 mm$**以上.

# 阳台
阳台按其与外墙面的关系分为**悬挑阳台**、**凹阳台**、**半挑半凹阳台**等几类.
